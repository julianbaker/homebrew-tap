# julianbaker/tap

Personal Homebrew tap.

```sh
brew install julianbaker/tap/furl
```

| Cask | What it is |
|---|---|
| [`furl`](Casks/furl.rb) | [Furl](https://github.com/julianbaker/Furl) — menu bar utility that makes off-screen overflow items reachable |

Cask versions and checksums are updated automatically by Furl's release pipeline.
